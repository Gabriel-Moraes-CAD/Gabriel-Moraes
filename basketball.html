Basketball Scoreboard App


This is a Basketball Scoreboard app. It was built using Vanilla JavaScript, HTML, CSS and Bootstrap 5. This app will keep track of scores, periods and time (12 minutes). A very straightforward user interface with simple options to keep track of your basketball game. I'd like to leave a special thanks to Per Harald Borgen and the Scrimba platform for walking me through the creation of this app. If you'd like to see the code, please visit my Github account.
